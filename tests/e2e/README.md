# E2E 测试用例

## 完整分析流程测试

### 测试场景
1. 用户上传 CSV 文件
2. 用户提交分析请求
3. Agent 执行分析
4. 返回结果

### 测试步骤
1. 启动后端服务
2. 启动前端服务
3. 访问前端页面
4. 上传测试 CSV 文件
5. 提交分析请求
6. 等待 Agent 执行完成
7. 验证结果显示

## 意图识别测试

### 测试场景
1. 测试关键词匹配
2. 测试 LLM 分类
3. 测试置信度计算

### 测试步骤
1. 启动后端服务
2. 发送不同类型的请求
3. 验证意图识别结果

## 文件上传测试

### 测试场景
1. 测试有效文件上传
2. 测试无效文件上传
3. 测试大文件上传

### 测试步骤
1. 启动后端服务
2. 上传各种类型的文件
3. 验证上传结果

## WebSocket 连接测试

### 测试场景
1. 测试连接建立
2. 测试消息传输
3. 测试断线重连

### 测试步骤
1. 启动后端服务
2. 建立 WebSocket 连接
3. 发送消息
4. 模拟网络断开
5. 验证重连成功
